# React Chart - Palette

The Chart component with the [Palette](../reference/palette.md) plugin allows you to specify a custom color scheme for a series.

## Basic Setup

Use the `Palette` plugin's `scheme` property to specify a Chart's color scheme. The following example shows how to change chart colors using the d3 library.
